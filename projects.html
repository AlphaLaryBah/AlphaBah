Finance Web App

Project Details

Finance: A Responsive Finance Web App that is a financial calculator/tracker dashboard for personal finance and stock data. This project is built with Create React App to build a static site. The static site is hosted on Netlify and pulled into GitHub for version control. The stylesheets are done using CSS, while some other visual components are built using React Bootstrap and MUI. The JavaScript coding style is enforced with a modified version of eslint Airbnb. I also use React-Prop-Types to maintain coding quality.

stocks data: fetched from Alpha Vantage: Free Stock APIs in JSON

Testing: Using React-testing-library and Jest.js for testing

Charts: are created using D3.js, React-Apexcharts.js, and Recharts.js

WalletGyde

Project Details

I built and implemented all technical aspects of the algorithm for a matchmaking feature. Users and Financial Coaches are matched based on their responses to specific questions as they are onboarded. Users and Coaches are notified of the matching results and their next steps through the Node-mailer module. Within the Microservices Architecture contained in Docker, I worked on legacy code. Additionally, I built the PostgreSQL database where profile information and responses of Users and Coaches are stored according to data models. I constructed the logic of the matchmaking algorithm in the Express Js server and worked with both Front-end and Back-end technologies to accomplish this deliverable.

I utilized the following Tech Stack:(Microservices Architecture, Docker, PostgresSQL, Sequelize, React Js, Gatsby Js, Javascript, Bootstrap, and Node-mailer).

I have built and improved upon numerous other features at WalletGyde and I continue to do so.
